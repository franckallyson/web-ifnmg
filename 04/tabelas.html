Seção de Tabelas

Nesta seção, iremos aprender um pouco mais sobre tabelas

Para criar uma tabela utilize a tag < table >

Cada linha da tabela é representada pela tag < tr >, e o valor de cada célula é representado pela tag < td >

Para definir uma borda utilize o parametro border="1", por exemplo.

Você Também pode criar uma linha de cabeçalho, utilizando a tag < th > ao invés da < td > na primeira linha!

Existem também tags estruturais da tabela, são elas: < thead >, < tbody >, < tfoot >.

Representam respectivamente, o cabeçalho, corpo e rodapé da tabela.

Existe a possibilidade de uma célula ocupar mais de uma coluna, fazemos isso através do parametro "colspan". Com a mesma ideia, podemos fazer uma célula ocupar duas linhas, através do parâmetro "rowspan".
